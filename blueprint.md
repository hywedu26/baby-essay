# Blueprint: 아기 성장 에세이 앱

## 1. 개요

이 애플리케이션은 부모가 아기와의 소중한 일상을 기록하고, 이 기록들을 바탕으로 AI를 통해 감성적인 육아 에세이를 생성, 저장 및 관리할 수 있도록 돕는 웹 앱입니다. Firebase를 기반으로 안전한 인증, 데이터 저장, 파일 관리를 구현합니다.

## 2. 핵심 기능 및 디자인 (현재 버전)

### 가. 인증 및 개인화
- **로그인:** 사용자는 닉네임과 프로필 사진을 등록하여 로그인합니다. 이 정보는 `localStorage`에 저장되어 재방문 시 자동으로 로그인됩니다.
- **개인화된 환영 메시지:** 메인 화면에 사용자의 프로필 사진과 닉네임이 표시됩니다.

### 나. 육아 기록 관리
- **기록 작성:** 날짜, 텍스트, 사진(미리보기만 가능)을 포함한 일일 기록을 작성하고 저장합니다.
- **기록 저장:** 모든 기록은 `localStorage`에 저장되어 영구적으로 보존됩니다.
- **기록 목록:** 작성된 기록들이 시간 순서대로 메인 화면에 표시됩니다. 각 기록은 수정 및 삭제가 가능합니다.

### 다. AI 에세이 생성
- **생성 요청:** 'AI 에세이 만들기' 버튼을 누르면 저장된 모든 기록을 Google AI (Gemini)에 전송합니다.
- **AI 프롬프트:** 아이를 사랑하는 부모의 관점에서 감성적인 에세이를 작성하도록 AI에게 지시합니다. 결과물은 HTML 형식입니다.
- **결과 표시:** 생성된 에세이는 모달 창을 통해 사용자에게 표시됩니다. 생성 중에는 로딩 메시지가 나타납니다.

### 라. 디자인 및 스타일
- **화면 구성:** 로그인 화면과 메인 화면으로 구성된 SPA(Single Page Application) 구조입니다.
- **색상 팔레트:** 따뜻하고 부드러운 파스텔 톤(#fde2e4, #dfe7fd, #fcf6bd, #e2f0d9)을 사용하여 포근한 느낌을 줍니다.
- **폰트:** 'Gaegu' 웹 폰트를 사용하여 손글씨 같은 아기자기한 느낌을 강조합니다.
- **레이아웃:** 모바일 반응형 디자인을 적용하여 모든 기기에서 최적화된 화면을 제공합니다. 중앙 정렬 레이아웃과 카드 형태의 UI를 사용하여 안정감을 줍니다.
- **상호작용:** 버튼 호버 효과, 부드러운 화면 전환 등 미묘한 애니메이션을 통해 사용자 경험을 향상시켰습니다.

## 3. 다음 개발 계획: Firebase 기반 인증 및 프로필 기능 강화

### 가. 목표
- 기존의 간단한 로그인 방식을 Firebase Authentication을 사용한 안전한 아이디/비밀번호 기반 인증 시스템으로 전환합니다.
- 사용자가 프로필 사진을 직접 관리하고, 아기의 D-day를 확인할 수 있는 개인화된 프로필 기능을 추가합니다.

### 나. 세부 실행 계획
1.  **Firebase 프로젝트 설정:**
    *   Firebase를 프로젝트에 연동하고, Authentication, Firestore, Storage 서비스를 활성화합니다.
    *   `.idx/mcp.json` 파일에 Firebase 서버 설정을 추가합니다.

2.  **HTML 구조 변경 (`index.html`):**
    *   **로그인/회원가입 화면:** 아이디(이메일), 비밀번호 입력 필드와 회원가입 버튼이 포함된 새로운 로그인 폼을 디자인합니다.
    *   **프로필 관리 섹션:**
        *   사용자 프로필 사진, 닉네임, 아기 생년월일 입력 필드를 포함하는 프로필 영역을 추가합니다.
        *   프로필 사진을 변경할 수 있는 UI(파일 업로드, 기본 이미지 선택)를 구현합니다.
        *   아기의 D-day(예: D+743)가 표시될 공간을 마련합니다.

3.  **JavaScript 로직 개편 (`main.js`):**
    *   **Firebase SDK 연동:** Firebase SDK를 ES6 모듈 방식으로 가져와 초기화합니다.
    *   **인증 로직 구현 (Firebase Auth):**
        *   `createUserWithEmailAndPassword`를 사용하여 이메일, 비밀번호로 회원가입하는 기능을 구현합니다.
        *   `signInWithEmailAndPassword`를 사용하여 로그인하는 기능을 구현합니다.
        *   `onAuthStateChanged`를 사용하여 사용자의 로그인 상태를 실시간으로 감지하고 화면을 전환합니다.
    *   **데이터 관리 로직 변경 (Firestore):**
        *   기존 `localStorage`에 저장하던 모든 데이터(육아 기록, 에세이, 사용자 정보)를 사용자 UID와 연결하여 Firestore에 저장하고 관리하도록 변경합니다.
    *   **프로필 관리 로직 구현:**
        *   **프로필 사진:** 사용자가 이미지를 업로드하면 Firebase Storage에 저장하고, 해당 이미지의 URL을 사용자의 프로필 정보(Firestore)에 업데이트합니다.
        *   **D-day 계산:** Firestore에 저장된 아기 생년월일을 가져와 오늘 날짜와의 차이를 계산하여 화면에 표시하는 함수를 작성합니다.
