# 육아 에세이 웹앱 개발 Blueprint

## 1. 프로젝트 개요

이 프로젝트는 아기 사진과 기록을 바탕으로 육아 에세이를 생성, 편집, 관리할 수 있는 웹 애플리케이션을 개발하는 것을 목표로 합니다. 사용자의 글쓰기 스타일을 학습하여 개인화된 결과물을 제공하며, 사용자와 상호작용하는 현대적이고 아름다운 UI를 제공합니다.

## 2. 핵심 기능

- **로그인 및 개인화:**
    - 아이디와 비밀번호를 이용한 사용자 인증.
    - 로그인 시, 등록된 아기 프로필 사진 또는 가족 사진으로 사용자 환영.
- **육아 기록 및 에세이 생성:**
    - 특정 날짜를 선택하여 아기 사진 업로드 및 간단한 기록 작성.
    - 업로드된 사진과 기록을 기반으로 AI가 육아 에세이 초안 생성.
    - 생성된 에세이를 사용자가 자유롭게 편집 및 저장.
- **AI 톤앤매너 학습:**
    - 초기 설정 시, 사용자가 기존에 작성한 육아 에세이 3편을 제출.
    - AI가 제출된 글의 톤앤매너(문체, 감성, 어휘 등)를 학습하여 이후 에세이 생성에 반영.
- **날짜 선택 기능:**
    - 기본 작성일은 현재 날짜로 설정.
    - 사용자가 원하는 과거 날짜를 선택하여 기록을 추가하거나 수정 가능.

## 3. 기술 스택 및 디자인

- **프론트엔드:** HTML, CSS, JavaScript (ES Modules)
- **백엔드 (서버리스):** Cloudflare Workers
- **AI:** Google Gemini
- **디자인:**
    - **Aesthetics:** 모던하고 따뜻한 느낌의 디자인을 적용하여 사용자에게 편안하고 즐거운 경험을 제공합니다.
    - **Layout:** 모바일과 웹에서 모두 완벽하게 작동하는 반응형 레이아웃을 구현합니다.
    - **Interactivity:** 사용자 경험을 향상시키는 미묘한 애니메이션과 인터랙티브 요소를 적용합니다.
    - **Accessibility:** 웹 접근성(A11Y) 표준을 준수합니다.

## 4. 개발 계획

- **1단계: 기본 구조 및 로그인 화면 구현 (완료)**
- **2단계: 육아 기록 및 사진 업로드 기능 구현 (완료)**
- **3단계: AI 에세이 생성 UI 구현 (완료)**
- **4단계: 서버리스 AI 프록시 구현 (완료)**

- **5단계: 실제 AI 서비스 연동 (진행중)**
    - **API 키 보안 설정:**
        - Cloudflare Pages 대시보드에 Google AI API 키를 `GEMINI_API_KEY`라는 이름의 비밀 환경 변수(Secret)로 등록. **(완료)**
    - **`functions/generate-essay.js` 수정:**
        - 기존의 Mock 함수를 실제 AI API 호출 로직으로 대체.
        - 프론트엔드에서 받은 기록들을 조합하여 AI에게 전달할 프롬프트(명령어) 생성.
        - 환경 변수에서 API 키를 안전하게 불러와 Google Gemini API에 에세이 생성을 요청.
        - AI가 생성한 결과물을 파싱하여 프론트엔드에 JSON 형태로 전달.

- **6단계 (예정):** 톤앤매너 학습 기능 구현.

- **7단계 (예정):** 에세이 편집 및 저장 기능 구현.
