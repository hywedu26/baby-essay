
# Blueprint: 아기 성장 에세이 앱

## 1. 개요

이 애플리케이션은 부모가 아기와의 소중한 일상을 기록하고, 이 기록들을 바탕으로 AI를 통해 감성적인 육아 에세이를 생성, 저장 및 관리할 수 있도록 돕는 웹 앱입니다.

## 2. 핵심 기능 및 디자인 (현재 버전)

### 가. 인증 및 개인화
- **로그인:** 사용자는 닉네임과 프로필 사진을 등록하여 로그인합니다. 이 정보는 `localStorage`에 저장되어 재방문 시 자동으로 로그인됩니다.
- **개인화된 환영 메시지:** 메인 화면에 사용자의 프로필 사진과 닉네임이 표시됩니다.

### 나. 육아 기록 관리
- **기록 작성:** 날짜, 텍스트, 사진(미리보기만 가능)을 포함한 일일 기록을 작성하고 저장합니다.
- **기록 저장:** 모든 기록은 `localStorage`에 저장되어 영구적으로 보존됩니다.
- **기록 목록:** 작성된 기록들이 시간 순서대로 메인 화면에 표시됩니다. 각 기록은 수정 및 삭제가 가능합니다.

### 다. AI 에세이 생성
- **생성 요청:** 'AI 에세이 만들기' 버튼을 누르면 저장된 모든 기록을 Google AI (Gemini)에 전송합니다.
- **AI 프롬프트:** 아이를 사랑하는 부모의 관점에서 감성적인 에세이를 작성하도록 AI에게 지시합니다. 결과물은 HTML 형식입니다.
- **결과 표시:** 생성된 에세이는 모달 창을 통해 사용자에게 표시됩니다. 생성 중에는 로딩 메시지가 나타납니다.

### 라. 디자인 및 스타일
- **화면 구성:** 로그인 화면과 메인 화면으로 구성된 SPA(Single Page Application) 구조입니다.
- **색상 팔레트:** 따뜻하고 부드러운 파스텔 톤(#fde2e4, #dfe7fd, #fcf6bd, #e2f0d9)을 사용하여 포근한 느낌을 줍니다.
- **폰트:** 'Gaegu' 웹 폰트를 사용하여 손글씨 같은 아기자기한 느낌을 강조합니다.
- **레이아웃:** 모바일 반응형 디자인을 적용하여 모든 기기에서 최적화된 화면을 제공합니다. 중앙 정렬 레이아웃과 카드 형태의 UI를 사용하여 안정감을 줍니다.
- **상호작용:** 버튼 호버 효과, 부드러운 화면 전환 등 미묘한 애니메이션을 통해 사용자 경험을 향상시켰습니다.

## 3. 다음 개발 계획: 에세이 저장 및 관리 기능

### 가. 목표
- 사용자가 AI를 통해 생성된 에세이를 저장하고, 저장된 에세이 목록을 언제든지 다시 볼 수 있는 기능을 추가합니다.

### 나. 세부 실행 계획
1.  **HTML 구조 변경 (`index.html`):**
    *   메인 화면에 '육아 기록'과 '저장된 에세이'를 전환해서 볼 수 있는 탭 UI를 추가합니다.
    *   저장된 에세이가 표시될 새로운 컨테이너 영역 (`<div id="essays-container">`)을 생성합니다.

2.  **JavaScript 로직 추가 (`main.js`):**
    *   **에세이 저장 기능:** '이 에세이 저장하기' 버튼에 클릭 이벤트 리스너를 추가합니다.
        *   클릭 시, 모달 창에 있는 에세이 내용과 생성 날짜를 객체 형태로 묶어 `localStorage`의 'essays' 키에 배열로 저장합니다.
    *   **에세이 목록 표시 기능:**
        *   앱 로드 시 `localStorage`에서 저장된 에세이 목록을 불러옵니다.
        *   각 에세이를 카드 형태로 디자인하여 '저장된 에세이' 탭 컨테이너에 표시합니다. 각 카드에는 삭제 버튼을 포함합니다.
    *   **탭 기능:** '육아 기록'과 '저장된 에세이' 탭을 클릭하면 해당하는 컨테이너는 보여주고 다른 컨테이너는 숨깁니다.
    *   **에세이 삭제 기능:** 각 에세이 카드에 있는 삭제 버튼 클릭 시, 해당 에세이를 `localStorage`와 화면에서 제거합니다.
